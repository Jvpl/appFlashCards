FLUXO DE NAVEGAÇÃO COMPLETO:

1. App Inicia com AppContent
   └─► Carrega NavigationContainer com AppTheme

2. User vê Bottom Tab Navigator (3 abas)
   └─► Padrão: "Início" ativa

3. Clicando em "Início":
   ├─► Verifica se teclado está visível
   ├─► Se visível: bloqueia clique
   ├─► Se invisível: renderiza DrawerNavigator
   │   └─► Renderiza HomeStackNavigator (com 10 telas)

4. Clicando em "Progresso" ou "Loja":
   ├─► Desativa DrawerNavigator
   ├─► Renderiza tela correspondente com animação Fade
   └─► Botão voltar volta para "Início"

5. Drawer Menu (lado direito):
   ├─► Ativável via swipe (apenas em DeckList)
   ├─► Ou via ícone no header
   ├─► Opções:
   │   ├─► Início (HomeDrawer/HomeStackNavigator)
   │   └─► Configurações (SettingsScreen)
   └─► Posicionado à direita (75% da largura)

6. Stack Navigation (dentro do Home):
   ├─► Telas:
   │   ├─► DeckList (raiz)
   │   ├─► SubjectList
   │   ├─► Flashcard
   │   ├─► AddDeck
   │   ├─► EditDeck
   │   ├─► AddSubject
   │   ├─► ManageFlashcards
   │   ├─► EditFlashcard
   │   ├─► EditSubject
   │   └─► FlashcardHistory
   └─► Transições: Fade (300ms)
